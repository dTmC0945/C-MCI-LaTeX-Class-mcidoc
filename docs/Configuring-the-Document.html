2 Configuring the Document ¶

The entire configuration is handled by a single command \SetMCI which the end user can change the behaviour by changing specific keys defined within. The accesible options changes depending on the state the mcidoc is in whereas some options are global and are accesible across all states of mcidoc.cls.
