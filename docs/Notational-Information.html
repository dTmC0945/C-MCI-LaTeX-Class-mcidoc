2.1 Notational Information ¶

Before options are described, it is worth explaining the slight notational quirks the documentclass has.

Majority of the options given by the \SetMCI are by default either as set or unset.

  • If a feature is set by default, then the option will be executed as long as the user does not alter its value.
  • Conversely, if a \SetMCI value is unset, then it will not show in the document or alter anything within the document as long as the user does not change its behaviour.
